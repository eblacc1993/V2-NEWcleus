# NEWCLEUS Phase 9

A deployable Next.js starter for the V2 Official NEWCLEUS operating system.

## Local run

```bash
npm install
npm run dev
```

## Netlify

Build command:

```bash
next build
```

Publish directory: leave blank for standard Next.js on Netlify.

## Suggested environment variables

- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY
- NEXTAUTH_SECRET
